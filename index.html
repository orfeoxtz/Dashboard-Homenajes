Manual Interno

1. Columnas recomendadas

Fecha, Gestor, Tipo_Homenaje, Tipo_Excedente, Valor, Sede y Observación.

2. Seguridad

El acceso visual valida clave y correos configurados. Para seguridad total se requiere backend o Apps Script con validación real.

3. Metas

La meta se calcula proporcionalmente según días seleccionados. También puedes configurar metas por gestor, sede, categoría y mes.

4. Pareto 80/20

Identifica qué gestores, servicios o sedes concentran el mayor porcentaje de ventas.

5. Reportes

PDF ejecutivo, Excel, CSV y backup JSON.

6. Antes de presentar

Revisa Diagnóstico API, Auditoría, Comparativo Anual, Pareto y Cierre Gerencial.
